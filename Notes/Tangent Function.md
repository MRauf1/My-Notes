# Definition

> [!info] Definition 1 (Tangent [[Function]])[^1]
> Tangent function is
> $$
> \begin{align}
> \tan(\theta) = \frac{\sin(\theta)}{\cos(\theta)}
> \end{align}
> $$
> provided $\cos(\theta) \neq 0$.

Has [[Domain]] $\{\theta | \theta \neq \frac{\pi}{2} + \pi k, k \in \mathbb{Z}\}$ and [[Range]] $(-\infty, \infty)$.

For a [[Right Triangle]] as depicted below, $\tan(\theta) = \frac{b}{a}$. Memorized as [[SOH-CAH-TOA]].
