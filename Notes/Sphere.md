# Definition

Sphere is a 3D version of a [[Circle|circle]] with radius $r$.[^1]

# Properties

## [[Surface Area|Surface Area]]

$$S = 4 \pi r^2$$

## [[Volume|Volume]]

$$V = \frac{4}{3} \pi r^3$$
