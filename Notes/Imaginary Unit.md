# Definition

> [!info] Definition 1 (Imaginary Unit)[^1]
> The imaginary unit, denoted as $i$, is $\sqrt{-1}$ that satisfies the following definitional properties
> 1) $i^2 = -1$
> 2) $c \in \mathbb{R}, c \geq 0 \implies \sqrt{-c} = i \sqrt{c}$
