# Definition

> [!info] Definition 1 (Set of Integer Multiples)
> For a given [[Integer]] $a$, this is a [[Set]] of $a$'s [[Multiple|multiples]]
> $$
> \begin{align}
> a\mathbb{Z} = \{an | n \in \mathbb{Z}\}
> \end{align}
> $$
> Some denote it as $\mathbb{Z}a$
