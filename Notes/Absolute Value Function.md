# Definition

> [!info] Definition 1 ([[Absolute Value|Absolute Value]] [[Function|Function]])[^1]
> $$
> \begin{align}
> f(x) = |x| =
> \begin{cases}
> -x & x < 0 \\
> x & x \geq 0
> \end{cases}
> \end{align}
> $$
> The [[Domain|domain]] is $[-\infty, \infty]$ and the [[Range|range]] is $[0, \infty]$.
