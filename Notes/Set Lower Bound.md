# Definition

> [!info] Definition 1 ([[Set]] Lower Bound)[^1]
> For a set $S$, if there exists a [[Real Number]] $m$ such that $s \geq m$ for all $s \in S$, then $m$ is the lower bound of $S$ and $S$ is bounded below.
