# Definition

> [!abstract] Theorem 1 ([[Limit of Function]] [[Function Composition]] Law)[^1]
> If $f$ is a function with limit $L$ (finite) and $g$ is a function defined on $\{f(x)\} \cup \{L\}$ and is [[Continuous Function]] at $L$, then $\lim_{x \rightarrow a} (g \circ f)(x) = g(L)$.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=169)
