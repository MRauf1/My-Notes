# Definition

> [!info] Definition 1 (Decreasing [[Function]])[^1]
> For a [[Function|function]] $f$ on an [[Interval Notation|interval]] $I$, $f$ is decreasing on $I$ $\iff$ $f(a) > f(b)$ for all [[Real Number|real numbers]] $a, b$ in $I$ with $a < b$.
