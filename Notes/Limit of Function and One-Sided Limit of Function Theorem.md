# Definition

> [!abstract] Theorem 1 ([[Limit of Function]] and [[One-Sided Limit of Function]])
> $$
> \begin{align}
> \lim_{x \rightarrow a} f(x) = L \iff \lim_{x \rightarrow a^-} f(x) = L = \lim_{x \rightarrow a^+} f(x)
> \end{align}
> $$

Limit exists if and only if $f(x)$ approaches the same limit no matter which direction $x$ approaches $a$.

> [!abstract] Theorem 2 ([[Limit of Function]] and [[One-Sided Limit of Function]] for [[Real-Valued Function]])
> If $f(\mathbf{x}) \rightarrow L_1$ as $\mathbf{x} \rightarrow \mathbf{a}$ along a path $C_1$ and if $f(\mathbf{x}) \rightarrow L_2$ as $\mathbf{x} \rightarrow \mathbf{a}$ along a path $C_2$ and $L_1 \neq L_2$, then the limit does not exist.

Limit exists if and only if $f(x_1, \dots, x_n)$ approaches the same limit no matter which direction $(x_1, \dots, x_n)$ approaches $(a_1, \dots, a_n)$.

Since there are infinite amount of such paths if the [[Domain]] is at least 2-dimensional, we can only check the above.
