# Definition

> [!info] Definition 1 ([[Function]] [[Minimum]])
> For a [[Function|function]] $f$, the value $b$ is the minimum $\iff$ $b \leq f(x)$ $\forall x$ in the [[Domain|domain]] of $f$.[^1]
