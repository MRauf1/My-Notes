# Definition

> [!info] Definition 1 (Horizontal [[Asymptote]])[^1]
> [[Line]] $y = c$ is a horizontal asymptote for a [[Function]] $y = f(x)$ if either
> 1) $\lim_{x \rightarrow -\infty} f(x) = c$, or
> 2) $\lim_{x \rightarrow \infty} f(x) = c$
