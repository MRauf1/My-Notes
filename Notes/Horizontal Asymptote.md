# Definition

> [!info] Definition 1 (Horizontal [[Asymptote]])[^1]
> [[Line]] $y = c$ is a horizontal asymptote for a [[Function]] $y = f(x)$ if as $x \rightarrow \infty$ or $x \rightarrow -\infty$, $f(x) \rightarrow c$
