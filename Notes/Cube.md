# Definition

Cube is a [[Rectangular Solid|rectangular solid]] with the same length, width, and height. Each side has length $s$.[^1]

![[Cube.png]]

# Properties

## [[Surface Area|Surface Area]]

$$S = 6s^2$$

## [[Volume|Volume]]

$$V = s^3$$
