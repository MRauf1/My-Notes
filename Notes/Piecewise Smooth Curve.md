# Definition

> [!info] Definition 1 ([[Piecewise]] [[Smooth Curve|Smooth]] [[Curve]])
> A [[Curve]] $\gamma$ is piecewise smooth (or a contour) if there is a [[Partition]] $t_0 = a < t_1 < \dots < t_{n - 1} < t_n = b$ of $[a, b]$ so that $\gamma$ is [[Continuous Function]] [[Differentiable Function]] on each $[t_i, t_{i + 1}]$.
