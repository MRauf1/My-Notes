# Definition

> [!info] Definition 1 (Function Parity)
> [[Function]] $f$ is
> - Even (Symmetric about the y-[[Axis|axis]]) $\iff$ $f(-x) = f(x)$
> - Odd (Symmetric about the [[Origin|origin]]) $\iff$ $f(-x) = -f(x)$

Whether a function is even, odd, or neither.
