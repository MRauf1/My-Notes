# Definition

> [!info] Definition 1 (Standard [[Logistic Function]])
> A type of logistic function where $L = 1, k = 1, x_0 = 0$ giving the function
> $$
> \begin{align}
> f(x) &= \frac{1}{1 + e^{-x}} \\
> &= \frac{e^x}{e^x + 1} \\
> &= \frac{e^{x/2}}{e^{x/2} + e^{-x/2}}
> \end{align}
> $$
