# Definition

> [!abstract] Theorem 1 ([[Uniformly Continuous Function]] [[Extension of Function]] Theorem)[^1]
> [[Function]] $f$ on $(a, b)$ is [[Uniformly Continuous Function]] on $(a, b)$ if and only if it can be extended to a [[Continuous Function]] $\tilde{f}$ on $[a, b]$.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=159)
