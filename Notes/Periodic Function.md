# Definition

> [!info] Definition 1 (Periodic [[Function]])[^1]
> [[Function]] $f$ is periodic if there exists $c \in \mathbb{R}$ such that
> $$
> \begin{align}
> f(t + c) = f(t)
> \end{align}
> $$
> for all $t \in \mathbb{R}$ where $t$ is in the function's [[Domain]].
> The smallest positive number $c$ for which this is true is called the [[Period]] of $f$.
