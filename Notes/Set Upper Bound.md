# Definition

> [!info] Definition 1 ([[Set]] Upper Bound)[^1]
> For a set $S$, if there exists a [[Real Number]] $M$ such that $s \leq M$ for all $s \in S$, then $M$ is the upper bound of $S$ and $S$ is bounded above.
