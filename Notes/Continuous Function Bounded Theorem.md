# Definition

> [!abstract] Theorem 1 ([[Continuous Function]] [[Bounded Function]])[^1]
> If $f$ is a [[Continuous Function]] on closed interval $[a, b]$, then $f$ is a [[Bounded Function]]. There exist [[Minimum]] and [[Maximum]] values $x_{min}, x_{max}$ in this interval.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=144)
