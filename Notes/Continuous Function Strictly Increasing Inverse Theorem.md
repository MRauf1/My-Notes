# Definition

> [!abstract] Theorem 1 ([[Continuous Function]] Strictly [[Increasing Function]] [[Inverse Function]] Theorem)[^1]
> Let $f$ be [[Continuous Function]] strictly increasing function on interval $I$. With $f(I) = J$, $f^{-1}$ is [[Continuous Function]] strictly increasing function on $J$.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=148)
