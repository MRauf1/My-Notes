# Definition

For arbitrary [[Set]] $A, B, C$
- $\emptyset \subseteq A$
- $A \subseteq A$
- $A \subseteq B \land B \subseteq C \implies A \subseteq C$ ([[Transitive]])
- $A = B \iff A \subseteq B \land B \subseteq A$
- $A \subseteq B \iff B^c \subseteq A^c$
- $A \setminus B = A \cap B^c$
