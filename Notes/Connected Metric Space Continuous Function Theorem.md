# Definition

> [!abstract] Theorem 1 ([[Connected Metric Space]] [[Continuous Function]] Theorem)[^1]
> Given [[Metric Space]] $(S_1, d_1), (S_2, d_2)$ and $f: S_1 \rightarrow S_2$ a [[Continuous Function]], if $E$ is a [[Connected Metric Space]] [[Subset]] of $S_1$, then $f(E)$ is a [[Connected Metric Space]] [[Subset]] of $S_2$

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=191)
