# Definition

> [!info] Definition 1 ([[Isaac Newton]] First Law)[^1]
> An object at rest will remain at rest and an object in motion will remain in motion unless acted on by an external [[Force]].
> [[Momentum]] is constant unless acted upon by an external [[Force]].
> If the net force acting on a body is 0, then there exists a set of [[Reference Frame]] (called [[Inertial Frame]]) in which that body has no [[Acceleration]].

[^1]: [Physics, Volume 1](zotero://open-pdf/library/items/5QBK4P96?page=59)
