# Definition

Theory of [[Vision|vision]] (325 BC) that provides the first mathematical explanation for vision using [[Geometry|geometry]]. [[Light|Light]] [[Ray|rays]] travel in straight lines and form a cone starting from the [[Eye|eyes]].

Gives rise to geometric reasoning and analysis of the world using geometry. For example, one can calculate the sizes of objects using geometry.

Originated by [[Euclid|Euclid]].[^1]
