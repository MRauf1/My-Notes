# Definition

- $\int_{\gamma} (c_1 f(z) + c_2 g(z)) dz = c_1 \int_{\gamma} f(z) dz + c_2 \int_{\gamma} g(z) dz$ for $c_1, c_2 \in \mathbb{C}$
- $\gamma_1 = \gamma|_{[a, c]}, \gamma_2 = \gamma|_{[c, b]} \implies \int_{\gamma} f(z) dz = \int_{\gamma_1} f(z) dz + \int_{\gamma_2} f(z) dz$
- $\int_{- \gamma} f(z) dz = - \int_{\gamma} f(z) dz$
