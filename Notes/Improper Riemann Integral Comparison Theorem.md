# Definition

> [!abstract] Theorem 1 ([[Improper Riemann Integral]] Comparison Theorem)[^1]
> If $f, g$ are [[Continuous Function]] with $f(x) \geq g(x) \geq 0$ for $x \geq a$
> - If $\int_a^{\infty} f(x) dx$ is convergent, then so is $\int_a^{\infty} g(x) dx$
> - If $\int_a^{\infty} f(x) dx$ is divergent, then so is $\int_a^{\infty} g(x) dx$
> Similar theorem can be constructed for the interval $(-\infty, a]$

[^1]: [Calculus: Early Transcendentals](zotero://open-pdf/library/items/EEFDQ9Y5?page=565)
