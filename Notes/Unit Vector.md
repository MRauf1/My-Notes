# Definition

> [!info] Definition 1 (Unit [[Vector]])[^1]
> [[Vector]] $\mathbf{v}$ is a unit vector if it has a [[Norm]] of $1$.
> In other words, $\mathbf{v} \cdot \mathbf{v} = 1$.
