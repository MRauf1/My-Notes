# Definition

> [!info] Definition 1 (Nowhere [[Dense Subset]])[^1]
> [[Subset]] $E$ of $S$ is nowhere dense in $S$ if its [[Closure Set]] $E^-$ has empty [[Interior Point|interior]].
> Alternatively, $E$ is nowhere dense if and only if $E^-$ is nowhere dense.
> Alternatively, $E$ is nowhere dense if and only if $S \setminus E^-$ is [[Dense Subset]] of $S$.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=182)
