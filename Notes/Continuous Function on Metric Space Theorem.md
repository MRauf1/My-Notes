# Definition

> [!abstract] Theorem 1 ([[Continuous Function]] on [[Metric Space]] Theorem)[^1]
> [[Function]] $f: S_1 \rightarrow S_2$ is continuous on $S_1$ if and only if $f^{-1}(U)$ is [[Open Set]] [[Subset]] of $S_1$ for every [[Open Set]] [[Subset]] $U$ of $S_2$.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=178)
