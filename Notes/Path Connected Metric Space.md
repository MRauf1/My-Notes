# Definition

> [!info] Definition 1 (Path Connected [[Metric Space]])[^1]
> [[Subset]] $E$ of [[Metric Space]] $(S, d)$ is path connected if for each pair $s, t \in E$, there exists a [[Continuous Function]] [[Curve]] $\alpha: [a, b] \rightarrow E$ such that $\alpha(a) = s, \alpha(b) = t$
