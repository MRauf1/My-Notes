# Definition

> [!info] Definition 1 ([[Metric]] Space)[^1]
> Metric space is the pair $(S, d)$ where $S$ is a [[Set]] and $d$ is a [[Metric]] on $S$.

# Properties
- For any [[Set]] $S$, $S = S^{\circ} \cup \partial S \cup S^e$, where $S^{\circ}$ is [[Interior Set]], $\partial S$ is [[Boundary Set]], $S^e$ is [[Exterior Set]].
