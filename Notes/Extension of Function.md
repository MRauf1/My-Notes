# Definition

> [!info] Definition 1 (Extension of [[Function]])[^1]
> [[Function]] $\tilde{f}$ is an extension of [[Function]] $f$ if $dom(f) \subseteq dom(\tilde{f})$ and $f(x) = \tilde{f}(x)$ for all $x \in dom(f)$.

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=157)
