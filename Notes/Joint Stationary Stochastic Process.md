# Definition

> [!info] Definition 1 (Joint [[Stationarity]] [[Stochastic Process]])
> Two [[Time Series]] $X_t, Y_t$ are jointly stationary if
> 1) $X_t, Y_t$ are each [[Stationary Stochastic Process]]
> 2) Cross [[Covariance]] [[Function]] is a function of only the lag $h$:
> 	1) $\gamma_{xy}(h) = Cov[X_{t + h}, Y_t]$
