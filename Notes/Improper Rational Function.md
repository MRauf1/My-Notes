# Definition

> [!info] Definition 1 (Improper [[Rational Function]])[^1]
> [[Rational Function]] $r(x) = \frac{p(x)}{q(x)}$ is improper if $deg(p) \geq deg(q)$. To make it proper, one must use [[Polynomial Division]].

[^1]: [Calculus: Early Transcendentals](zotero://open-pdf/library/items/EEFDQ9Y5?page=525)
