# Definition

- $\sum_{i=1}^n c = nc$
- $\sum_{i=1}^n c a_i = c \sum_{i=1}^n a_i$
- $\sum_{i=1}^n (a_i + b_i) = \sum_{i=1}^n a_i + \sum_{i=1}^n b_i$
- $\sum_{i=1}^n (a_i - b_i) = \sum_{i=1}^n a_i - \sum_{i=1}^n b_i$
