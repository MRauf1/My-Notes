# Review[^1]

Excellent book for a computer scientist to learn the basics behind how computers work. It is really in depth and quite lengthy. Practically though, one should mainly understand the optimizations that can help a software engineer, like parallelism, caching, prefetching, and so on.

[^1]: [Computer Organization and Design: The Hardware/Software Interface](zotero://open-pdf/library/items/YWPB5EDC?page=1)
