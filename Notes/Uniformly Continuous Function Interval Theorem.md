# Definition

> [!abstract] Theorem 1 ([[Uniformly Continuous Function]] Interval Theorem)[^1]
> Let $f$ be [[Continuous Function]] on interval $I$. Let $I^{\circ}$ be interval obtained by removing from $I$ any endpoints that are in $I$. If $f$ is [[Differentiable Function]] on $I^{\circ}$ and $f'$ is [[Bounded Function]] on $I^{\circ}$, then $f$ is [[Uniformly Continuous Function]].e

[^1]: [Elementary Analysis: The Theory of Calculus](zotero://open-pdf/library/items/GUY2WR3V?page=161)
