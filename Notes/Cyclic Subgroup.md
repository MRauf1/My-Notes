# Definition

> [!info] Definition 1 (Cyclic Subgroup)
> For a [[Group]] $G$, a [[Subgroup]] $H \leq G$, a subgroup is cyclic if $H = <\{a\}>$ for some $a \in G$

Subgroup is cyclic if it is a [[Subgroup Generated by Subset]] consisting of a single element.
