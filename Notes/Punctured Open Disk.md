# Definition

> [!info] Definition 1 (Punctured [[Open Disk]])
> Punctured open disk centered at $z_0 \in \mathbb{C}$ with radius $r \in \mathbb{R}^+$ is $B^*(z_0, r) = \{z \in \mathbb{C} : 0 < |z - z_0| < r\}$

It is an open disk with the [[Point]] $z_0$ punctured out (removed).
