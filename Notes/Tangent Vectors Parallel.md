# Definition

> [!info] Definition 1 ([[Tangent Vector]] [[Parallel]])[^1]
> $2$ tangent vectors $\mathbf{v}_{\mathbf{p}}, \mathbf{w}_{\mathbf{q}}$ are parallel if they have the same vector part ($\mathbf{v} = \mathbf{w}$).
