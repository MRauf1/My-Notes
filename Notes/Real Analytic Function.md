# Definition

> [!info] Definition 1 ([[Real Number]] [[Analytic Function]])
> [[Analytic Function]] defined on an [[Open Set]] of [[Set of Real Numbers]]. It has a [[Power Series]] that converges to $f(x)$ within some [[Interval Notation|interval]] $(x_0 - r, x_0 + r)$, and it is [[Infinitely Differentiable Function]].
