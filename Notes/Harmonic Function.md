# Definition

> [!info] Definition 1 (Harmonic [[Function]])
> Let $D \subseteq \mathbb{C}^n$ be [[Open Set]]. [[Function]] $f: D \rightarrow \mathbb{R}$ is harmonic if $f$ has [[Continuous Function]] 2nd order [[Partial Derivative]] on $D$ and if the [[Laplacian Operator]] $\triangle f = f_{xx} + f_{yy} = 0$ on $D$.
