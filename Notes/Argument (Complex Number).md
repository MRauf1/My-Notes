# Definition

> [!info] Definition 1 (Argument of [[Complex Number]])
> See [[Complex Number in Polar Coordinates]]

# Properties
- $arg(\overline{z}) = - arg(z)$
- $arg(z_1 z_2) = arg(z_1) + arg(z_2)$
- $arg(\frac{z_1}{z_2}) = arg(z_1) - arg(z_2)$
- $arg(z^n) = n\ arg(z)$
- $Arg(z) = arg(z) + 2 k \pi$, where $k \in \mathbb{Z}$ such that the result is in $(- \pi, \pi]$
