# Definition

> [!info] Definition 1 ([[Absolute Value|Absolute Value]] [[Equality|Equality]] Properties)[^1]
> For $x, c \in \mathbb{R}$
> $$
> \begin{align}
> |x| = 0 &\iff x = 0 \\
> \text{For}\ c > 0, |x| = c &\iff x = c \lor -x = c \\
> \text{For}\ c < 0, |x| = c\ &\text{has no solution}
> \end{align}
> $$
