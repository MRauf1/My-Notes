# Definition

> [!info] Definition 1 ([[Harmonic Function]] [[Conjugate]] [[Function]])
> Let $D \subseteq \mathbb{C}^n$ be [[Open Set]] and $u, v$ be [[Harmonic Function]] on $D$. $v$ is harmonic conjugate of $u$ if $u + iv$ is [[Analytic Function]] on $D$, or equivalently, if $u_x = v_y$ and $u_y = - v_x$ on $D$.

# Properties
- If $D$ is a [[Domain]] and $v_1, v_2$ are harmonic conjugates of $u$, then $\exists c \in \mathbb{R}$ such that $v_1 = v_2 + c$.
