# Definition

- $b^a = c \iff log_b(c) = a$
- $log_b(1) = 0$
- $log_b(u) = log_b(w) \iff u = w$ for all $u, w \in \mathbb{R^+}$
- $log_b(b^x) = x$ for all $x \in \mathbb{R}$
- $b^{log_b(x)} = x$ for all $x > 0$
