# Definition

> [!info] Definition 1 ([[Absolute Value]] of [[Complex Number]])
> For $z = x + iy$,
> $$
> \begin{align}
> |z| = \sqrt{x^2 + y^2}
> \end{align}
> $$

Can be thought of as [[Magnitude]].

# Properties
- $|Re(z)| = |x| \leq |z|$
- $|Im(z)| = |y| \leq |z|$
- $|\overline{z}| = |z|$
- $|z|^2 = z \overline{z}$
- $|z_1 z_2| = |z_1| |z_2|$
- $|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$
- $|z^n| = |z|^n$
