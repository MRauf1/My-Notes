# Definition

> [!info] Definition 1 (Closure Set)[^1]
> Closure $E^-$ of a [[Set]] $E$ is the [[Set Intersection]] of all [[Closed Set]] containing $E$.

# Properties
- [[Closure Set Basic Properties]]
- $E^- = E \cup \partial E$, where $\partial E$ is [[Boundary Set]]
