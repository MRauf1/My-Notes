# Definition

> [!info] Definition 1 ([[Plane]] [[Curve]])[^1]
> Let $\beta$ be a [[Unit Speed Curve]] on $\mathbb{R}^3$ with $\kappa > 0$. Then $\beta$ is a plane curve if and only if $\tau = 0$.

A curve is a plane curve if it lies in a single plane.
