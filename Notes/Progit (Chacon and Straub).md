# Review[^1]

Excellent book for beginners to version control system as it clearly explains how Git works.

A lot of the sections contain in depth information that may not be useful on a daily basis, so I would recommend reading Git Basics and Git Branching chapters in depth, while leaving the rest of the chapters as a reference that one can come back to if they need that material.

[^1]: [ProGit](zotero://open-pdf/library/items/Y53JUJJW?page=1)
