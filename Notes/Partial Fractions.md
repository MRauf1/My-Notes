# Definition

> [!info] Definition 1 (Partial Fractions)[^1]
> Given a [[Rational Function]], to express the fraction as a sum of decomposed linear and/or quadratic denominators (which is possible due to the [[Real Factorization Theorem]])
> 1) Make it [[Proper Rational Function]]
> 2) The form is:
> 	1) Linear terms (with repetition) = $\frac{A_1}{(a_1 x + b_1)} + \dots + \frac{A_n}{(a_n x + b_n)^n}$
> 	2) Quadratic terms (with repetition) = $\frac{B_1x + C_1}{(a_1 x^2 + b_1 x + c_1)} + \dots + \frac{B_nx + C_n}{(a_n x^2 + b_n x + c_n)^n}$

[^1]: [Calculus: Early Transcendentals](zotero://open-pdf/library/items/EEFDQ9Y5?page=525)
