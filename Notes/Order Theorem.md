# Definition

> [!info] Definition 1 (Order Theorem)
> For [[Group]] $G$ and $H \leq G$, if $|G| = n < \infty$ and $g \in G$, then $o(g) = |<g>|$ [[Divides]] $|G|$
> In other words, $o(g) = |<g>|$ and $|G|$ cannot be [[Relatively Prime]]
