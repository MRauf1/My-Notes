# Definition

> [!info] Definition 1 (Score [[Function]])
> Given a [[Log-Likelihood Function]] $L(\beta)$, the score function is
> $$
> \begin{align}
> u(\beta) = \frac{\partial L(\beta)}{\partial \beta}
> \end{align}
> $$
