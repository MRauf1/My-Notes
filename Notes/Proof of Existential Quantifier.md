# Definition

> [!info] Definition 1 (Proof of Existential Quantifier)
> For a [[Proposition|statement]] $\exists x P(x)$, to prove it
> 1) Let $x$ be a specific value for which $P(x)$ is true
> 2) Prove $P(x)$
