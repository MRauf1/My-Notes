# Properties

## Definition
Let $a \in G, n \in \mathbb{Z}$, then $a^n$ is defined as
	- $a^0 = e, a^1 = a, a^{-1} =$ inverse of $a$
	- $n \geq 1 \implies a^{n+1} = a^n a$
	- $n \leq -1 \implies a^{n-1} = a^n a^{-1}$
## Power Properties
Let $m, n \in \mathbb{Z}$
	- $a^m a^n = a^{m + n}$
	- $(a^m)^n = a^{mn}$
