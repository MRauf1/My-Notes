# Definition

> [!info] Definition 1 (Convex [[Set]])[^1]
> [[Set]] $E$ is convex if
> $$
> \begin{align}
> x, y \in E \land 0 < t < 1 \implies tx + (1 - t)y \in E
> \end{align}
> $$

Set is convex if whenever $E$ contains 2 points, it also contains the [[Line Segment]] connecting them.
