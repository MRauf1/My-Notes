# Definition

> [!info] Definition 1 ([[Tangent Space]] [[Linear Isomorphism]])[^1]
> Tangent space $T_{\mathbf{p}}(\mathbb{R}^3)$ is linearly isomorphic to $\mathbb{R}^3$ with the function $\phi(\mathbf{v}_{\mathbf{p}}) := \mathbf{v}$.
