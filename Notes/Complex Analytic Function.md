# Definition

> [!info] Definition 1 ([[Complex Number]] [[Analytic Function]])
> [[Function]] $f$ on [[Open Set]] $U$ of $\mathbb{C}^n$ is analytic if it is [[Differentiable Function]] on $U$, has a [[Derivative]] that is [[Continuous Function]] on $U$, and has a [[Power Series]] that converges to $f(z)$ within some [[Disk]] $B(z_0, r)$.

# Properties
- [[Infinitely Differentiable Function]]
- [[Holomorphic Function]]
- The real and imaginary parts are [[Harmonic Function]]
- [[Complex Analytic Function Inverse]]
- [[Morera's Theorem]]
- [[Cauchy's Inequality]]
